

Oracle中SQL的执行计划

查询10号部门的员工
explain plan for select * from emp where deptno=10;
select * from table(dbms_xplan.display);

补充：Oracle数据库的优化器
	1、RBO：基于规则的优化器
	2、CBO：基本上都是CBO，基于成本的优化器

补充：Oracle中的索引的类型
	1、B树索引：默认
	2、位图索引
